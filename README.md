# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Sections Included**:
  - Hero section with profile picture and introduction
  - About me section with bio
  - Education timeline
  - Skills showcase
  - Projects portfolio
  - Contact form (non-functional demo)

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (ES6+)
- Google Fonts (Poppins)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information:
   - Update name, title, and bio in the HTML
   - Replace the placeholder profile image URL
   - Add your education details
   - Update skills and projects
   - Modify contact information

## Customization

### Profile Picture
Replace the placeholder image URL in `index.html`:
```html
<img src="your-image-url.jpg" alt="Profile Picture" id="profile-img">
```

### Colors
Modify the CSS variables in `styles.css` to change the color scheme:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... other colors */
}
```

### Content
Edit the HTML file to update:
- Personal information
- Education details
- Skills list
- Project descriptions
- Contact information

## Deployment

### GitHub Pages
1. Push your code to a GitHub repository
2. Go to Settings > Pages
3. Select your branch and save
4. Your site will be available at `https://yourusername.github.io/repository-name`

### Netlify
1. Sign up/login to Netlify
2. Drag and drop your project folder, or
3. Connect your GitHub repository
4. Your site will be automatically deployed
